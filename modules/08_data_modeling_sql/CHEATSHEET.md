# 08 — Cheat‑sheet rápido (MySQL)

- Primary key:
```sql
PRIMARY KEY (columna)
```

- Foreign key:
```sql
FOREIGN KEY (columna) REFERENCES otra_tabla(columna)
```

- Unique:
```sql
UNIQUE (columna)
```

- Check:
```sql
CHECK (condición)
```

- Index:
```sql
CREATE INDEX idx_nombre ON tabla(columna);
```
